Proyecto de Recetas donde puedes compartir tus recetas favoritas y también guardar tus recetas que mas te gusten.

### Lenguaje de programación, Frameworks y Librerias.

- **Php (Laravel 8)**
- **JavaScript (VueJs)**
- **OWL Carousel**
- **MomentJs**
- **JQuery**
- **Axios**
- **sweetalert2**
